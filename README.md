# sao-rollup-lib

> Scaffolding out a rollup lib.

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao rollup-lib my-project
```

### From git

```bash
sao asuka109/sao-rollup-lib my-project
```
